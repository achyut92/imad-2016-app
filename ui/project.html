MyQre - Clinic Locator

Android Application

A mobile app where user can locate clinics in Singapore based on different search criteria(such as current location,specific location,etc). User could also register with the clinic using the app and request for a Queue number.It can also provide directions to the clinic.

Client App - Sports/Event Monitoring System

Android Application

A mobile app used by the participants of a sports event(an obstacle race) organized by an event management company. The app lists all the upcoming, rgistered and past events of a registered app user. The user can check into an event via QR code at the venue and able to send location data to the command center(Web application) for monitoring purposes.
